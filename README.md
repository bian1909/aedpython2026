## Descripción general:
Sistema de Gestión de Gimnasio desarrollado en Python
El sistema permite administrar de manera total un gimnasio, incluyendo:
- **Gestión de socios**: altas, bajas, modificaciones y búsqueda
- **Control de membresías**: diferentes tipos con precios y descuentos
- **Administración de cuotas**: registro de pagos y control de deudores/morosos
- **Gestión de actividades**: creación de clases con cupos máximos
- **Inscripciones**: registrar a los socios en actividades y gestionar las mismas
- **Control de asistencia**: registro de entrada/salida por tipo (gimnasio, actividad, libre)
- **Control de errores:** posibles errores de parte del usuario se encuentran contemplados y parcheados.

El sistema utiliza archivos txt para el almacenamiento no volatil de datos y sigue los contenidos vistos en la cátedra.
